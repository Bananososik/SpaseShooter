PROJECT TITLE:
Fleet Arena (Greenfoot 2D Shooter)
PURPOSE OF PROJECT:
Create a 2D top-down space shooter with ship battles, two playable ships,
enemy waves, shooting mechanics, and a win/lose state.
VERSION or DATE:
08.04.2026
HOW TO START THIS PROJECT:
Open project.greenfoot and run MyWorld.
AUTHORS:
Ivan
USER INSTRUCTIONS:
Player 1: A/D/W/S to move, SPACE to fire.
Player 2: Arrow keys to move, ENTER to fire.
Destroy enemies to gain score. Reach 100 points to win.
If both player ships are destroyed, the game is over.
