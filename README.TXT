PROJECT TITLE:
Fleet Arena (Greenfoot 2D Shooter)
PURPOSE OF PROJECT:
Create a 2D top-down space shooter with ship battles, a single playable ship,
enemy waves, shooting mechanics, and a win/lose state.
VERSION or DATE:
08.04.2026
HOW TO START THIS PROJECT:
Open project.greenfoot and run MyWorld.
AUTHORS:
Ivan
USER INSTRUCTIONS:
Single player: A/D/W/S to move, SPACE to fire.
Destroy enemies to gain score and survive as long as possible.
Each 300 score points spawns a boss with multiple attack patterns, including ray attacks.
HP and Shield are shown as percentages in a vertical HUD column on the left.
Rare upgrade pickup (lvlUp) can drop from enemies; one is guaranteed before the first boss.
Upgrade pickup increases max HP and upgrades player bullet type from Bullet_1 up to Bullet_4.
Enemy ships now have multiple attack and movement patterns.
If your ship is destroyed, the game is over.
